asnake
======

Ascii_snake

compile with: 
	'cd src'
	'make'
 
probably you have to install ncurses-dev first.

snake control:

	w: up
	s: down
	a: left
	d: right

	p: pause
	
asnake
Released under the GNU GPL.
-s =SPEED        Set the speed of the snake, default is 5
-h               Show this help screen
